    • I wanted to take on a challenge, so I decided to sign up for the Daily Ui Newsletter. Each day I was faced with a new challenge to create a design based on an open ended prompt. With each of these challenges, I knew I wanted to try and show off a broad array of design aesthetics for different devices. Before I even opened up Sketch, I would jot down a few notes and did some rough sketches. I looked at Dribbble, Behance and Pinterest for inspiration. It was great having such simple prompts, and being able to interpret it a different way each day.
