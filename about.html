I decided to learn how to code at the New York Code + Design Academy where I fell in love with front end development. I was always more visually oriented, and I decided to learn more about UI/UX. After learning more about UI/UX I not only loved the visual design process, but I learned a lot about researching, conducting interviews, usability tests, prototyping Having both the technical background and design background, I am able to combine my skills and create innovative experiences.
